# guid
Human friendly GUID/UUID library

# Install
`pip install guid`

# Usage
```python
>>> from guid import GUID
>>> guid = GUID()
>>> guid.uuid
'd95b880b-fcec-47fb-bcc4-d49f71415cf8'
>>> guid.slug
'2VuIC_zsR_u8xNSfcUFc-A'
```
